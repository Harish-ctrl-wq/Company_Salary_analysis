# Company Salary Analysis (Pandas Project)

## Overview  
This project is a small but realistic data analysis exercise using **Python and Pandas**.  
The focus is not on fancy visuals, but on understanding how to **clean, group, sort, and analyze data** in a way that makes sense.

The dataset is simple on purpose â€” the goal was to practice **core data analysis thinking**, not to hide behind complexity.

---

## Dataset  
The dataset was created as a dictionary and converted into a Pandas DataFrame.

It contains basic employee information:
- Name  
- Salary  
- Department  
- Age  

This allowed me to practice:
- Grouping and aggregation  
- Sorting values  
- Choosing the right chart for the data  

---

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## Analysis Performed
Using Pandas, I performed the following steps:
- Grouped salary data by department  
- Calculated total salary per department  
- Sorted salaries to find top earners  
- Visualized results using appropriate charts
